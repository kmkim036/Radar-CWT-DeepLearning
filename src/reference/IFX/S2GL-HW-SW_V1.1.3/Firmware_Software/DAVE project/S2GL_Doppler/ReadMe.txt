S2GL_Doppler

===============================================================================
Description of project
===============================================================================
- Motion detection Firmware including Micrium µC/Probe™ GUI for Demo Sense2GoL board.
- Use this project in case you are interested in motion, speed, and direction of 
  movement detection of moving objects.

===============================================================================
Features
===============================================================================
- Motion is indicated via the three on-board LEDs:
  * Red LED: Target approaching Radar sensor
  * Orange LED: Target moving away from Radar sensor
  * Blue LED: No motion
- Motion detection can be configured via Threshold and min/max velocity settings
- I/Q raw data is streamed via UART interface
- Matlab communication interface for data extraction
