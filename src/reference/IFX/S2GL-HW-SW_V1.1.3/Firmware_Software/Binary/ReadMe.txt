===============================================================================
Folder Contents
===============================================================================

- S2GL_Doppler.hex
  *  Precompiled FW project that provides implementation of Doppler algorithm 
     to measure velocity and motion of a single target (strongest one)

- S2GL_Doppler.elf
  *  Precompiled compiling and linking process output file (ELF file), containing the name, 
     data type and address of all firmware global variables.
     This file can be parsed by the µC/Probe™ project.
